# NexoIA — Sistema de Automatización de Contenido

**Tu presencia digital en automático. Tú solo apruebas.**

## Cómo funciona

1. **La IA genera contenido** (posts, carruseles, stories, reels)
2. **Tú apruebas o rechazas** desde el Panel de Aprobación
3. **Se publica automáticamente** en @nexoia_cali

## Estructura

```
content/
├── pending/     → Contenido nuevo esperando tu aprobación
├── approved/    → Contenido que aprobaste → se publica
└── published/   → Contenido ya publicado

approval/        → Panel web para aprobar/rechazar
config/          → Configuración de la cuenta
templates/       → Plantillas de contenido
```
